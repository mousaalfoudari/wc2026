# موقع توقعات كأس العالم 2026

موقع لتسجيل توقعات نتائج وهدافي مباريات كأس العالم 2026، مع نظام نقاط، الدبل، سؤال البونص، والجوكر، وجدول ترتيب عام.

بدون أي اعتماديات خارجية (Node.js فقط) — يعمل بقاعدة بيانات SQLite محلية.

**جدول دور المجموعات (٧٢ مباراة) يتحمّل أوتوماتيك عند أول تشغيل**، والنتائج تتحدث تلقائياً من مصدر خارجي مجاني (بدون أي مفتاح API) — المشتركون يسجلون ويتوقعون بنفسهم، بدون أي تدخل يدوي من الأدمن لإدخال النتائج. التفاصيل والقيود بالأسفل.

## التشغيل محلياً

يحتاج Node.js إصدار 22.5 أو أحدث.

```
npm start
```

يفتح على `http://localhost:3000`. أول تشغيل ينشئ حساب أدمن تلقائياً وتطبع بياناته في الطرفية (الاسم وكلمة المرور). تقدر تحدد اسم وكلمة مرور الأدمن بنفسك بدل العشوائي:

```
ADMIN_NAME=admin ADMIN_PASSWORD=كلمة_مرور_قوية npm start
```

## متغيرات البيئة (اختيارية)

- `PORT` — رقم البورت (افتراضي 3000).
- `DB_PATH` — مسار ملف قاعدة البيانات (افتراضي `data/app.db`).
- `ADMIN_NAME` / `ADMIN_PASSWORD` — بيانات حساب الأدمن الأول.
- `SESSION_SECRET` — مفتاح تشفير الجلسات. لو ما حددته، يتولّد تلقائياً ويُحفظ بملف محلي — لكن هذا الملف لا يبقى بعد كل رفعة جديدة على بعض الاستضافات، فيفضّل تحديده بنفسك في الإنتاج (راجع دليل الاستضافة).

## رفع الموقع للإنترنت

راجع ملف `دليل_الاستضافة.md` لخطوات الرفع على Render (حوالي 7$ بالشهر، استضافة موثوقة لا تمسح بيانات المشتركين).

## الجدول والنتائج الأوتوماتيكية

- **دور المجموعات (١٧ جولة، ٧٢ مباراة)** يتحمّل تلقائياً عند أول تشغيل للسيرفر — ما يحتاج الأدمن يضيف شي. لو الجولات موجودة بالفعل، التحميل يتجاهل نفسه (آمن يشتغل كل مرة السيرفر يبدأ من جديد).
- **النتائج** تتحدث تلقائياً كل ١٠ دقائق من مصدر بيانات عام مجاني ([openfootball/worldcup.json](https://github.com/openfootball/worldcup.json)) بدون أي مفتاح أو تسجيل. ⚠️ المصدر نفسه يتحدث تقريباً **مرة كل يوم** (يدوياً من القائم عليه)، فهو "تلقائي" بمعنى "ما يحتاج إدخال يدوي من الأدمن" — مو لحظي بالثانية أثناء المباراة.
- بلوحة `/admin` فيه زر **"تحديث النتائج الآن"** لو حبيت تتحقق فوراً بدون انتظار الدورة التلقائية.
- **أدوار خروج المغلوب** (ثمن النهائي وما بعده) ما تتحمل أوتوماتيك لأن أسماء الفرق ما تتحدد إلا بعد نهاية دور المجموعات — الأدمن يضيفها يدوياً من نفس لوحة التحكم لما تتضح الفرق المتأهلة.
- مطابقة اسم الهداف للنقاط الإضافية (+٢ نقطة) تعتمد على تطابق الاسم بالضبط (بعد تجاهل المسافات وحالة الأحرف) — يفضّل توصية المشتركين يكتبون الاسم الرسمي الكامل للاعب (مثلاً "Lionel Messi") عشان أعلى فرصة يطابق الاسم اللي بيجي من المصدر الخارجي.

## ملاحظات

- بيانات قاعدة البيانات تُخزّن بمجلد `data/` المحلي — هذا المجلد مستثنى من الرفع على GitHub عمداً (`.gitignore`) لأنه يحتوي بيانات حساسة وقواعد بيانات لا يجب نشرها على العلن.
- الأدمن يدير الجولات والمشتركين من لوحة `/admin` بعد تسجيل الدخول بحساب الأدمن، ويقدر يدخل أو يعدّل أي نتيجة يدوياً متى احتاج (مثلاً لتصحيح خطأ، أو لإضافة أدوار خروج المغلوب).
- زر **"تصدير الترتيب PDF"** بلوحة الأدمن (للأدمن فقط) يفتح صفحة طباعة نظيفة لجدول الترتيب وتشغّل نافذة الطباعة أوتوماتيك — الأدمن يختار "حفظ كـ PDF" منها، بدون أي اعتمادية خارجية.
